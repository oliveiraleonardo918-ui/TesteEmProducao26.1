# 📦 SIGE - Sistema de Gestão de Equipamentos

Sistema web para gerenciamento de empréstimos e devoluções de equipamentos acadêmicos.

---

## 🚀 Funcionalidades

- 🔐 Login administrativo
- 📊 Dashboard do professor
- 📦 Empréstimo de equipamentos
- 🔁 Histórico de devoluções
- 🧭 Interface moderna e responsiva

---

## 🛠️ Tecnologias

- Next.js
- TypeScript
- Tailwind CSS
- Lucide Icons
